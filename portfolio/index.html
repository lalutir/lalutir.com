World Cup Simulator

A Python pipeline that simulates the 2026 FIFA World Cup knockout phase one million times. Team strength comes from Elo ratings; a trained match-outcome model converts that into win / draw / loss probabilities; Monte Carlo simulation runs the full bracket out a million times to produce advancement odds for every team. The result renders as a static dashboard — no server-side logic once it's deployed, just generated HTML.

P2000 Reader

A live feed of Dutch emergency-services alerts — fire, ambulance, police — scraped from p2000-online.net and broadcast to every connected browser over WebSocket, with optional Web Push notifications. There's no database: the last 50 alerts live in memory on the FastAPI backend and in the browser's localStorage, which keeps the whole thing simple to run and easy to reason about.
